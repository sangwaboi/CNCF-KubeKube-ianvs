1. **Prerequisites**:
   - Python 3.11+ with conda
   - Ianvs framework installed
   - 8GB+ RAM, 4+ CPU cores

2. **Setup**:
   ```bash
   cd Industrial_Locomotion_Benchmark
   conda activate ianvs-locomotion
   ```
